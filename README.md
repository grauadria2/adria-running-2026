# adria-running-2026

Proyecto personal de entrenamiento running — Temporada 2026.

## Objetivos

| Carrera | Fecha | Objetivo |
|---------|-------|----------|
| Media maratón test | Agosto 2026 | Correr a 160-165 ppm, evaluar progreso |
| Media maratón PR | Noviembre 2026 | 1:35-1:40 (4:30-4:44 min/km) |
| Maratón | Diciembre 2026 | ~3:35-3:45, sin sufrir |

## Estructura

```
skills/          → Skill de running con metodología y perfil personal
plans/           → Plan de entrenamiento semana a semana
data/            → Exportaciones de Strava
logs/            → Registro semanal, MAF tests, carreras
```

## Cómo usar esto con Claude

1. Abre conversación nueva con Claude
2. Adjunta `skills/running-planner/SKILL.md` + el log que sea relevante
3. Pregunta lo que necesites — Claude tendrá todo el contexto

## Parámetros clave

- FC máx real: 205 ppm
- Zona aeróbica práctica: 150-158 ppm
- Volumen base (Mar 2026): ~20 km/semana
- Ritmo actual a FC baja: ~7:10 min/km
